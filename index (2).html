Ukuran Array
Pseudocode
procedure InsertionSort(A[0..n-1])
  // Urutkan array A secara ascending
  for i ← 1 to n - 1 do
    key ← A[i]       // simpan elemen kunci
    j ← i - 1
    while j ≥ 0 and A[j] > key do
      A[j + 1] ← A[j]  // geser elemen ke kanan
      j ← j - 1
    end while
    A[j + 1] ← key   // tempatkan key di posisi yang tepat
  end for
end procedure
Materi Insertion Sort

📖 Pengertian

Insertion Sort adalah algoritma pengurutan yang bekerja seperti cara manusia mengurutkan kartu. Setiap elemen baru "disisipkan" ke posisi yang tepat di antara elemen-elemen yang sudah terurut sebelumnya.

⚙️ Cara Kerja

  • Mulai dari elemen indeks ke-1
  • Simpan elemen tersebut sebagai key
  • Bandingkan key dengan elemen di kiri
  • Geser elemen yang lebih besar ke kanan
  • Sisipkan key ke posisi yang kosong
  • Ulangi sampai seluruh array terurut

✅ Keunggulan

  • Efisien untuk data kecil
  • Stabil (urutan elemen sama tidak berubah)
  • Adaptif — cepat untuk data hampir terurut
  • In-place (tidak butuh memori tambahan)
  • Mudah diimplementasikan

⚠️ Kelemahan

  • Lambat untuk data besar (O(n²))
  • Tidak efisien jika data acak sepenuhnya
  • Kalah dengan Quick Sort / Merge Sort untuk n besar
